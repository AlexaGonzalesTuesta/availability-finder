Import Calendar

  1. Click the button below to open Outlook Web → Shared Calendars settings
  2. Under "Publish a calendar", pick your calendar and set to "Can view all details"
  3. Click Publish
  4. Right-click the ICS link and select Save link as... to download the .ics file
  5. Drag it into the box below, or click to browse

Tip: Bookmark the ICS link. Next time, just click it to download a fresh copy, then drag it in. Takes 5 seconds.
